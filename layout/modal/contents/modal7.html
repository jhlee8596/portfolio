문구류 등 판매하는 모나미 홈페이지의 클론코딩 사이트입니다.

주요 기능 및 특징

  • 드롭다운 메뉴
  • 상품 이미지 슬라이드
  • 탭 메뉴
  • data-

사용 기술 및 언어 클릭 시 세부 내용들을 확인 할 수 있습니다.

  • Swiper-Slide Plugin

    이미지 슬라이드를 구현하기 위해 Swiper-Slide 플러그인을 활용하였습니다.

제작 과정

  • Swiper-Slide Plugin

    • Slick-Slide, Swiper-Slide 중 무엇을 사용 할 지 고민을 하였는데 원하는 기능을 구현하기에는 Swiper-Slide가 더 좋다고 판단하여 체택하였습니다.
    • 이미지 슬라이드가 6개 까지 모두 보이고 7번째 이미지가 조금 잘려서 보이게끔 구현해야 하는데 마침 Swiper-Slide에서 지원하는 키값 중 slidesPerView 에 양수 입력하여 조금 더 세밀하게 조정할 수 있었습니다.

  • Drop=Dwon Menu

    • JavaScript의 mouseenter, mouseleave, click 이벤트 리스너를 활용하여 메뉴에 마우스 오버 시 드롭다운되는 메뉴와, 토글버튼을 눌렀을 때 드롭다운 되고, 다시 토글하기 전 까지 드롭다운이 유지되는 메뉴를 구현하였습니다.

  • Tab-Menu

    • 카테고리의 버튼인 a태그의 title 속성을 JavaScript의 getAttribute() 메서드를 사용하여 각 카테고리별 슬라이드 박스에 부여한 data-category와 연결하여 탭메뉴를 구현하였습니다.

문제점 확인

  • Swiper-Slide

    • 4개의 슬라이드 박스를 사용해야 하는데 처음에는 4개의 박스에 모두 같은 객체를 선언하였는데 첫 번째 박스를 제외한 3개의 박스에는 슬라이드 기능이 작동하지 않았습니다.
      그래서 구글링을 통해 알아보았더니 컨테이너에 mySwiper 클래스가 들어가는 부분의 클래스명을 각각 다르게 설정하면 작동하는 것을 알아내어 구현하였습니다.
