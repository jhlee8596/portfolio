자격증 수강 및 발급이 가능한 '한국자격검정평가진흥원' 리뉴얼 홈페이지입니다.

* 프로젝트 이슈 - UI개발만 완료된 상태이기에 서버측 코드를 필요로 하는 정상적인 기능 작동은 하지 않습니다.

주요 홈페이지 및 특징

  • 강의를 선택 시 장바구니에 담아주는 기능
  • 현재 강의 진도율 표시
  • 합격 후기 작성 페이지
  • 결제 페이지 내 포인트 기능

사용 기술 및 언어 jQuery, JavaScript

  • Tab Menu

    한 페이지 내에 여러가지 정보를 담고 있기 때문에 Tab Menu를 통해 카테고리 분류 후 해당 카테고리에 맞는 상품을 배치하였습니다.

  • Progress Bar

    강의 페이지에 필수적으로 들어가는 학습 진도율을 circle-progress 라이브러리를 사용하여 인터페이스를 구현하였습니다.

  • 장바구니

    수강신청 페이지에 있는 장바구니 기능은 과목 선택 버튼을 클릭할 시 해당 과목의 이름, 가격을 불러오고 checkbox가 checked된 개수를 불러와 선택한 과목의 개수를 카운트하게 됩니다.

제작 과정

  • 협업

    • 사내 디자이너와 Figma툴을 사용한 협업을 통해 기획을 함께 하였고, 디자인의 피드백과 구현에 대한 이슈사항을 논의하였습니다.

  • 구글링

    • 간단한 인터렉션 UI, html / css 구조는 모두 직접 작업하였고, 처음 제작해보는 기능들은 구글링을 많이 참고하였습니다.

문제점 확인

  • header, footer include 불가

    • 기존 재사용이 많은 header와 footer 는 분리해놓고 php의 include를 통해 불러오는 방식이었는데 cafe24의 기간 만료로 인해 다시 html파일로 변형한 뒤 붙여놓았습니다.
