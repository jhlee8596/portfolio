현대자동차 캐스퍼 메인 홈페이지입니다.

주요 기능 및 특징

  • jQuery Cookie - 팝업창 (오늘 하루 안 보기)
  • SwiperSlide Plugin - 이미지 슬라이드
  • Math() - 랜덤 배너 이미지

사용 기술 및 언어

  • jQuery Cookie

    광고 팝업창은 사용자가 원하지 않는 정보일 경우 보이지 않게 하는것이 좋습니다. localStorage를 활용하여 코드를 작성할 수도 있겠지만 jQuery에서 지원하는 쿠키 기능을 활용해 보았습니다.

  • Math()

    항상 같은 배너 이미지만 보여진다면 제공할 수 있는 정보의 양이 헌저히 떨어질 것이라 생각합니다. slide를 활용할 수도 있지만 Math() 함수를 활용한 랜덤 배너 이미지도 구현 해보았습니다.

작업 화면 이미지 클릭 시 크게 볼 수 있습니다. (작업화면이 현재와 다를 수 있습니다.) * 저작권 이슈가 있는 경우 첨부하지 않았습니다.
